ABOUT

This is a coding contest based on arrays,strings, algorithms, data structures and problem solving. As contest name says, it is not only a coding contest and also to test their quickness of brain and fingers in solving the codes in a limited time and you will be able to challenge yourself to push the limits.

DETAILS

  • This is a coding contest based on arrays, strings, algorithms, data structures, and problem-solving. The participants need to solve a coding problem in a given stipulated time.
  • There are a total of 15 problems and all problems have the same points allotted to them. Participants are ranked according to the most problems solved. The tie will be compensated according to the time consumed to solve the problems.
  • Discussing FlashCode’s problems or any aspect of the problem, on any other platform on the web, on identification, could lead to disqualification. Almost all languages are supported in this Flash Code contest.
  • Participants will be provided 60 minutes in which 15 questions want to be attended
    • Don’t forget to put on your seat belts, this is going to be fast.
